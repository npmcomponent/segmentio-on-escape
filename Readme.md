
# on-escape

  Attach an escape key handler.

## Installation

    $ component install segmentio/on-escape

## API

### onEscape(fn)
  Bind a handler `fn(event)` to be called when the `ESC` key is pressed.

### .unbind(fn)
  Unbinds a handler `fn`.

### .bind(fn)
  For symmetry.
